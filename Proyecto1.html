Proyecto 2: Programación 2

Descripción

Este proyecto implementa un sistema en C# y ASP.NET con apoye de una base de datos en SQL Server, cuenta con capas de aplicación, negocio y datos. Incluye un módulo de préstamos con tablas repetidoras, un login con manejo de cookies y el uso de MasterPages para mantener consistencia en la interfaz. Aunque es un proyecto muy simple, cuenta con una buena base para por ejemplo escalar el proyecto a aplicaciones reales, cubre en general un sistema de prestamos en una biblioteca. Ademas de se de los primeros trabjaos en trabajo en equipo en el que se coordino muy bien las distribuciones de tareas.

Tecnologías

  • C#
  • ASP.NET
  • CSS
  • T-SQL

Proyecto Tabloid

Descripción

Este proyecto desarrolla un sistema para controlar un dispositivo humidificador. Se compone de dos áreas principales: un sistema Arduino para la parte electrónica y una aplicación/interfaz para manipular el dispositivo. Es un proyecto creado con otros compañero, la idea principal era un dispositvo pequeño tipo humificador, pero con extras, por ejemplo configurar recordatorios que se avisaban por luces o sonidos, ademas la aplicacion deberia usarse en Moviles para tener acceso y manipularlo siempre que quisieramos, al trabajar con IOT se podria mantener un control en tiempo real gracias a los servidores de AdaFruit.

Tecnologías

  • C
  • Python
  • C++
  • Arduino

A continuación esta el repositorio con el respaldo del proyecto, si se conecta a AdaFruit y se crean los Feeds necesarios el proyecto podria funcinar. Aun debo crear subir los esquemas electronicos y lo necesario en AdaFruite para que funciones sin problema.

Proyecto Godot

Descripción

Este proyecto utiliza el motor Godot Engine para la creación de videojuegos. Incluye escenas, scripts en GDScript y recursos gráficos que permiten experimentar con mecánicas de juego y prototipos interactivos. El proposito principal es mas un proyecto persona que continuara creciendo, algunas veces hago tutoriales en youtube para crear videojuegos en Godot, y utiliso este proyecto para respaldar y compartir todo el contenido de esos tutoriales. Aque de momento es algo sensillo y no suelo hacerlo muy seguido, es tambien el respaldo de la creación de juegos que siempre me ha gustado

Tecnologías

  • Godot Engine
  • GDScript
  • Escenas (.tscn)
  • Assets gráficos y de sonido
